# Simple Beam Simulation

## Overview
Simulation analysis of a simple beam created in PTC Creo Parametric as part of the CodeAlpha Mechanical Design Internship (Task 2).

## Tools Used
- PTC Creo Parametric

## Contents
- `design_files/` — Creo part files (.prt)
- `simulation/` — Simulation setup and results
- `images/` — Result plots and screenshots
- `report/` — Task report (PDF/Word)

## Description
A simple beam was modeled in Creo Parametric and subjected to simulation analysis to evaluate its structural behavior (e.g., deflection, stress distribution) under applied loading conditions.
